## Deployment

This site is deployed using GitHub Pages.

### To Enable GitHub Pages:

1. Go to your repository: https://github.com/Guri8003/Guri8003
2. Click on **Settings**
3. Scroll down to **Pages** (in the left sidebar)
4. Under **Source**, select **main** branch
5. Click **Save**
6. Your site will be live at: https://guri8003.github.io/Guri8003/
